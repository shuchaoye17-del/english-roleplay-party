# Expression Card System

English learning content should become collectible game assets.

Each expression card includes:

- English expression
- Chinese meaning
- Usage scene
- Emotion type
- Example sentence
- Card rarity
- Unlock source
- Review status

Example card:

- Expression: No worries.
- Chinese: 没关系 / 不用担心。
- Card type: 缓和气氛卡
- Usage: Use when someone apologizes or makes a small mistake.
- Emotion: Friendly, relaxed, comforting.
- Example sentence: No worries. It happens.

## Rules

- Each roleplay scenario should reward 1 to 3 expression cards.
- Cards should reappear in later scenarios.
- Cards should be usable as hints or voice prompt options.
- The user should feel they are collecting useful English tools, not memorizing a word list.
- Cards should look like game props, not flashcards from a textbook.
