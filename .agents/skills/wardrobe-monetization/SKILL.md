# Wardrobe Rules

Cosmetic items should support identity, status, and social display.

Allowed visual items:

- Outfits
- Avatar frames
- Chat bubbles
- Result effects
- Titles
- Stickers
- Profile backgrounds

Do not sell gameplay advantages:

- No higher English scores
- No easier scoring
- No required progress advantage

Visual items should not affect scoring.

## Unlock Timing

- Stage 7: unlock wardrobe preview
- Stage 14: friends can see outfit and title
- Stage 21: unlock shop preview
- Stage 30: unlock season preview

MVP:

- Implement wardrobe UI with free items only
- Locked items should show future unlock copy
- Do not implement payment yet
