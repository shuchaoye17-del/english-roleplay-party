# Bright Party UI Style

The app should feel bright, social, energetic, and playful.

## Color Direction

Use:

- White / warm off-white backgrounds
- Coral pink for fun actions
- Bright yellow for rewards
- Sky blue for social / communication
- Soft purple for roleplay / magic moments
- Fresh green for success / ready states
- Dark navy only for text contrast

Avoid:

- Dark cinematic style
- Black-gold premium style
- Heavy brown vintage style
- Low-energy beige-only layouts
- Traditional education UI
- SaaS dashboard feeling

## Layout Rules

- Mobile first.
- Use large rounded cards.
- Use character avatars prominently.
- Use playful badges.
- Use large CTA buttons.
- Use clear room status.
- Use celebratory result screens.
- Use icons like stage, microphone, chat bubble, role mask, stars, trophy, confetti.

## Typography

- Use bold, friendly headings.
- Keep body text short.
- Use short Chinese labels.
- English expressions should be readable and visually highlighted.

## Screen Rules

Avatar creation:

- Big character preview
- Style options
- Personality tags
- Clear “进入英语派对” CTA

Story lobby:

- Colorful story cards
- Solo story as the main entry
- Clear locked states for duo and party modes

Team room:

- Player avatars
- Role cards
- Ready state
- Room code
- AI stand-in slot

Roleplay:

- Large scene / character area
- Dialogue bubbles
- Role/emotion reminder
- Big microphone button

Result:

- Big score
- Fun title
- MVP / personal title
- Rewards
- Expression cards
- Share button
