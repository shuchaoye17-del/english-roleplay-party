# UI Style Guide

## Style Keywords

Bright, youthful, social, playful, party-game, voice-first, roleplay, expressive, collectible.

## Color Direction

- Background: white / warm off-white
- Primary: coral pink
- Reward: bright yellow
- Social: sky blue
- Roleplay: soft purple
- Success: fresh green
- Text: dark navy / slate

Avoid dark cinematic pages as the main visual direction.

## Layout

- Mobile first
- One primary action per screen
- Large rounded cards
- Big tappable buttons
- Short text
- Strong visual feedback
- Character avatars visible often

## Components

### Primary Button

Large, rounded, colorful, playful.

### Story Card

Should include:

- Scenario title
- Location / mood
- Recommended players
- Estimated time
- Reward preview

### Mic Button

The most important gameplay control.

Should be:

- Large
- Bottom aligned
- Easy to tap
- Animated during recording

### Role Card

Should show:

- Role identity
- Emotion target
- Hidden mission
- Turn status

### Result Card

Should feel celebratory:

- Big score
- Title badge
- Reward cards
- Confetti / stars

## Copy Tone

Use theatrical and game-like language:

- “轮到你登场啦”
- “按住说台词”
- “剧场助演正在帮你接一句”
- “演绎完成”
- “获得表达卡”

Avoid school-like language:

- “开始考试”
- “做题”
- “语法错误”
- “口语作业”
