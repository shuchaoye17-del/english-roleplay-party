# Unlock Roadmap

The product must support solo play from the beginning. Multiplayer is an advanced unlock, not the first barrier.

## Stages

### Stage 1–3: Solo Story Basics

- Create avatar
- Play solo stories
- Learn voice turn flow
- Earn first expression cards

### Stage 4–6: AI Partner Mode

- AI teammate becomes more visible
- User learns role, emotion, mission, and feedback
- Lobby previews duo mode

### Stage 7: Duo Story Unlock

Unlock copy:

“你已经准备好和朋友一起演英文小剧场了。”

Features:

- Invite one friend
- 2-player roleplay
- AI narrator
- Duo result card

### Stage 14: Trio Story Unlock

Features:

- 3-player rooms
- More roles
- Teamwork score
- MVP title

### Stage 21: Wardrobe Shop Preview

Features:

- Friends can see outfits
- Locked cosmetic items become visible
- No payment in MVP

### Stage 30: Season Preview

Features:

- Season themes
- Rare outfits
- Scenario badges

## MVP Unlock Rules

For the first version, implement locked UI states only:

- Solo story: available
- Duo story: locked until stage 7
- Party mode: locked until stage 14
- Wardrobe shop: locked until stage 21

No real payment or real progression system is required in the first MVP.
